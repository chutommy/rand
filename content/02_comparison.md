## Comparison

### `math/rand`

* excellent interface
* rich method set
* easy to use
* not cryptographically random (results can be easily regenerated)

### `crypto/rand`

* cryptographically secure random number generator
* hard to use
* limited interface

### `chutommy/rand`

* beautiful math's rand interface
* nice and rich method set
* simple and easy to use
* true RNG
